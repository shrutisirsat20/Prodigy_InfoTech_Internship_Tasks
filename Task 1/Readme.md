# Task 1: Data Visualization using Excel

## Objective
To create a bar chart or histogram to visualize the distribution of a categorical or continuous variable (such as age or gender) using the given dataset.

## Tools Used
- Microsoft Excel

## Dataset
The dataset was provided as part of the internship task and contains population-related data suitable for visualization.

## Work Done
- Studied and understood the dataset
- Identified categorical/continuous variables
- Created a bar chart in Excel
- Analyzed the distribution shown in the chart

## Outcome
This task helped in understanding basic data visualization concepts and how charts can be used to represent data clearly and effectively.
